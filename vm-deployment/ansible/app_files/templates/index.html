Dummy Monitoring App

This is a simple Flask application that exposes Prometheus metrics for monitoring demonstrations.

Available Endpoints:

Home Slow Request Error Request Prometheus Metrics

Metrics Information:

  • app_request_count: Counts the number of requests by method, endpoint, and HTTP status
  • app_request_latency_seconds: Measures request latency by method and endpoint
  • app_error_count: Counts errors by method, endpoint, and error type
